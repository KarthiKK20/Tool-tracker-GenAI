# GenAI Trend Tracker

A data pipeline and analytics dashboard that tracks trending GenAI tools across platforms.

## Features

- Web scraping from GitHub and Reddit
- ETL pipeline using Python and Pandas
- SQLite database storage
- Trend scoring algorithm
- Interactive analytics dashboard (Flask + Chart.js)
- Leaderboard and weekly growth visualization

## Tech Stack

Python
Selenium
BeautifulSoup
Pandas
SQLite
Flask
Chart.js
